\chapter{Wprowadzenie}

Strona internetowa \textbf{"VEXILLARIUS"} stworzona została na potrzeby drugiego projektu (Zakres \textbf{JavaScript}) z przedmiotu \textbf{Technologie Internetowe} na 2 roku Informatyki na \textbf{Uniwersytecie Rzeszowskim}. Strona jako projekt posiada skrypty napisane w języku \textbf{JavaScript}, które nadają witrynie pewne funkcjonalności. Całość spełnia podstawowe standardy w tworzeniu stron internetowych, którymi między innymi są \textbf{responsywność}, intuicyjny układ, jak i również płynne działanie umieszczonych na stronie skryptów.\newline

The website \textbf{"VEXILLARIUS"} was created for the second project (JavaScript) in the \textbf{Internet Technologies} course for second-year Computer Science students at the \textbf{University of Rzeszów}. The project includes scripts written in \textbf{JavaScript}, which provide the site with certain functionalities. The entire project meets basic standards for website development, including \textbf{responsiveness}, an intuitive layout, and the smooth operation of scripts placed on the site.
